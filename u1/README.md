# Installing
## Tools required
```
sudo apt-get install git maven
```
 
## Getting the code
```
git clone https://github.com/totalorder/id2209
```

## Getting dependencies
```
cd id2209/ 
./install_jade.sh
```

## Compiling
```
mvn package
```

# Running
## Starting the system
```
./u1/run.sh
```
## Starting lookup agent
```
./u1/run_dflookup.sh
```
